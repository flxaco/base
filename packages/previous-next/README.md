# @pnx-mixtape/previous-next

previous-next component.

![previous-next  component](https://previousnext.github.io/mixtape/screenshots/previous-next.png)

### Installation

```
npm install @pnx-mixtape/previous-next --save-dev
```

### How to use

**CSS**

- PostCSS workflow: `@import '@pnx-mixtape/previous-next';`
- Sass/Eyeglass: `@import 'mixtape-previous-next';`
- Native CSS: `@import url('dist/previous-next.css');`
- Link tag: `<link href="dist/previous-next.css" rel="stylesheet" type="text/css">`
