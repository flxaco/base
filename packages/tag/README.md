# @flxaco/tag

Tag component

### Installation

```
npm install @flxaco/tag --save-dev
```

### How to use

**CSS**

- PostCSS workflow: `@import '@flxaco/tag';`
- Native CSS: `@import url('dist/tag.css');`
- Link tag: `<link href="dist/tag.css" rel="stylesheet" type="text/css">`

### Developer notes

To use with default values:

```css
@import "@flxaco/tag";
```
