# @flxaco/logo

Logo component.

### Installation

```
npm install @flxaco/logo --save-dev
```

### How to use

**CSS**

- PostCSS workflow: `@import '@flxaco/logo';`
- Native CSS: `@import url('dist/logo.css');`
- Link tag: `<link href="dist/logo.css" rel="stylesheet" type="text/css">`

### Developer notes

To override variables in a project:

```css
@import "@flxaco/logo";
```
